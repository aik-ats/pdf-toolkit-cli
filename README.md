# PDF Toolkit (PDFツールキット)

PDFファイルの暗号化・復号、ページの抽出・分割・結合、画像への変換などを簡単に行える、コマンドライン（CLI）ベースのユーティリティツールです。
直感的なメニュー画面と、ドラッグ＆ドロップ（D&D）による簡単なファイル入力に対応しています。

## ✨ 機能一覧

1. **PDFをパスワードで暗号化する（手動／環境変数）**
   - パスワードを入力して暗号化します。未入力時は環境変数 `PDF_DEFAULT_PASSWORD` が使用されます。
2. **PDFをランダムパスワードで暗号化する**
   - 自動生成された強力なランダムパスワードで暗号化し、パスワードを画面に表示します。
3. **PDFのパスワードを解除する（復号）**
   - 現在のパスワードを入力して暗号化を解除します。
4. **PDFから特定のページを抽出する**
   - 「1,3,5-7」のようにページ番号を指定して抽出できます。
5. **PDFを分割する**
   - 指定したページ（例：「3,5」）を境目にして複数のPDFに分割したり、全ページを1ページずつのファイルに一括分割したりできます。
6. **PDFのページを逆順に並び替える**
   - 最終ページから1ページ目に向かって逆順に並び替えます。
7. **複数のPDFを1つに結合する**
   - 指定した複数のPDFファイルを1つのファイルにまとめます。
8. **PDFを画像に変換する**
   - PDFの全ページをPNGまたはJPEG画像として出力します。
9. **複数の画像を1つのPDFに変換する**
   - 複数枚の画像ファイルを順番通りに1つのPDFファイルとしてまとめます。

## 🚀 環境構築

本ツールは Python 3 と `PyMuPDF`、およびリッチなコンソール出力を提供する `rich` ライブラリを使用しています。

```bash
# リポジトリのクローン
git clone https://github.com/あなたのユーザー名/pdf-toolkit-cli.git
cd pdf-toolkit-cli

# 仮想環境の作成と有効化（Windows）
python -m venv venv
.\venv\Scripts\activate

# 必要なパッケージのインストール
pip install -r requirements.txt
```

## 📖 使い方

以下のコマンドでアプリを起動します。

```bash
python main.py
```

1. 画面にメニューが表示されますので、実行したい機能の番号を入力します。
2. 処理対象のファイルパスを聞かれますので、**エクスプローラーからPDFファイル（複数ある場合はまとめて）をドラッグ＆ドロップ** して Enter を押してください。
3. 機能に応じて、追加のパラメータ（パスワードやページ番号など）を入力すると処理が実行されます。

> **Note:**
> 暗号化の処理において、元のファイルはファイル名に `_uncrypted` が付与されてバックアップされ、暗号化済みのファイルが元のファイル名で出力されます。

## 📦 実行可能ファイル（.exe）のビルド方法

Python環境がインストールされていないPCで実行するために、単一の `.exe` ファイルにパッケージ化することができます。

付属のバッチファイルを実行してください：

```bash
.\build.bat
```

ビルドが完了すると `dist` フォルダが作成され、その中に `PDFTool.exe` が生成されます。このexeファイル単体でどのWindowsマシンでも動作させることができます。

## 🛡 セキュリティとプライバシー（完全ローカル動作）

本ツールは**完全なローカル環境（オフライン）**で動作するように設計・実装されています。
機密性の高い人事資料、財務データ、社外秘のPDFなどであっても安心してご使用いただけます。

*   **ネットワーク通信ゼロ**: PDFの解析、暗号化、画像変換などのすべての処理は、お使いのPCのCPUおよびメモリ上のみで行われます。外部サーバーへのデータの送信、テレメトリ（利用状況の送信）などは一切行われません。
*   **安全なライブラリ**: 使用している主要ライブラリ（`PyMuPDF`, `Rich`）はいずれも世界中で広く使われているオープンソースであり、バックグラウンドでネットワーク通信を行うような悪意のあるコードやトラッキング機能は含まれていません。

## 🛠 使用ライブラリ

*   [Rich](https://rich.readthedocs.io/): 美しいCLIインターフェースの構築
